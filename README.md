# Java POO

Repositório com exercícios e projetos desenvolvidos no âmbito da aprendizagem de **Programação Orientada a Objetos em Java**.

## Conteúdos
- Classes e Objetos
- Encapsulamento
- Métodos e Construtores
- Herança
- Classes Abstractas
- Polimorfismo
- Enums
- ArrayList
- Menus em consola

## Organização
O repositório encontra-se organizado por aulas, fichas práticas e exercícios temáticos.

## Execução
Cada exercício pode ser compilado e executado a partir da respetiva pasta:

```bash
javac *.java
java Main
